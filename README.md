## Syntax
### Training
To start the training:
```
python3 vae_gan.py
```
To resume training from a previous training instance:
```
python3 vae_gan.py continue
```

### Testing
To test the functionality of the model:
```
python3 vae_gan.py test
```

### Generation
```
python3 vae_gan.py generate
```
